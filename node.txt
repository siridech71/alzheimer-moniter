## ขั้นตอนการติดตั้ง

### ขั้นที่ 1 — Clone โปรเจกต์

```bash
git clone https://github.com/username/alzheimer-monitor.git
cd alzheimer-monitor
```

### ขั้นที่ 2 — สร้าง Conda Environment

```bash
conda create --name PROJECT_YOLO python=3.11
conda activate PROJECT_YOLO
```

### ขั้นที่ 3 — ติดตั้ง Library ทั้งหมด

```bash
pip install torch torchvision torchaudio
pip install ultralytics opencv-python requests numpy python-dotenv
pip install lap
pip install psycopg2-binary
```

### ขั้นที่ 4 — ตั้งค่าไฟล์ .env

```bash
cp .env.example .env
```

เปิดไฟล์ `.env` แล้วใส่ข้อมูลจริง

```
SMTP_HOST=smtp.gmail.com
SMTP_PORT=587
EMAIL_USER=อีเมลที่จะส่ง@gmail.com
EMAIL_PASS=xxxx xxxx xxxx xxxx
EMAIL_TO=อีเมลผู้ดูแล@gmail.com

PG_HOST=localhost
PG_PORT=5432
PG_USER=postgres
PG_PASS=รหัสผ่าน PostgreSQL
PG_DB=alzheimer_db
```

> หมายเหตุ: EMAIL_PASS ต้องเป็น App Password จาก Google ไม่ใช่รหัส Gmail ปกติ  
> ขอได้ที่ myaccount.google.com → Security → App passwords

### ขั้นที่ 5 — สร้าง Database ใน PostgreSQL

```bash
psql -U postgres -c "CREATE DATABASE alzheimer_db;"
```

หรือสร้างผ่าน pgAdmin โดย คลิกขวาที่ Databases → Create → Database → ตั้งชื่อ alzheimer_db → Save

### ขั้นที่ 6 — ตั้งค่า config.py

เปิดไฟล์ `config.py` แล้วแก้ค่าตามสภาพแวดล้อมจริง

```python
# กล้อง: 0 = webcam, หรือใส่ path วิดีโอ
CAMERA_SOURCE = 0
# หรือ
CAMERA_SOURCE = "video/myvideo.mp4"

# พิกัด Safe Zone (ปรับให้ตรงกับกล้องจริง)
# วิธีหาพิกัด: รันโปรแกรมก่อน ดูขนาดภาพที่ print ออกมา
# แล้วค่อยกำหนดพิกัดให้เหมาะสม
SAFE_ZONE_POLYGON = np.array([
    [100, 100],   # มุมบนซ้าย
    [860, 100],   # มุมบนขวา
    [860, 440],   # มุมล่างขวา
    [100, 440],   # มุมล่างซ้าย
], dtype=np.int32)
```

## การทำงานของระบบ

```
เปิดกล้อง
    ↓
YOLO ตรวจจับคน + ByteTrack กำหนด ID
    ↓
เช็คว่าจุดเท้าอยู่นอก Safe Zone ไหม
    ↓
ถ้าอยู่นอก → จับเวลา (ต้องนานกว่า 4 วินาที)
    ↓
ครบ 4 วินาที → บันทึกภาพ + บันทึก DB + ส่ง Email
```

## ดูข้อมูลใน Database

เปิด pgAdmin → alzheimer_db → Schemas → public → Tables → events  
คลิกขวาที่ events → View/Edit Data → All Rows

หรือดูผ่าน Python

```bash
python -c "from database import get_recent_events; [print(e) for e in get_recent_events()]"
```

## เทคโนโลยีที่ใช้

| Library | เวอร์ชัน | หน้าที่ |
|---|---|---|
| Python | 3.11 | ภาษาหลัก |
| ultralytics | 8.x | YOLOv8 ตรวจจับคน |
| opencv-python | 4.8.x | ประมวลผลภาพ |

## Hardware ที่รองรับ

- **Mac Apple Silicon (M1/M2/M3)** — ใช้ MPS GPU อัตโนมัติ
- **Windows/Linux + NVIDIA GPU** — ใช้ CUDA อัตโนมัติ
- **CPU ทั่วไป** — รันได้ แต่ช้ากว่า
